Electoral Vote Simulator Documentation

Project Overview

The Electoral Vote Simulator is a Python-based console application designed to simulate an electoral voting process between two candidates. The program allows users to cast their votes, displays the results in real-time, and provides an option to export the results to a CSV file for record-keeping.

Purpose

The primary purpose of this project is to simulate a voting process, making it an excellent tool for demonstrating Python programming skills. The project showcases the handling of user inputs, the application of conditional logic, and the ability to manage data dynamically. It's ideal for educational settings, interviews, or as a portfolio piece demonstrating practical coding abilities.

Key Features

1. Voting System

User Interaction: Users can cast their votes for either of the two candidates through a simple console interface.

Vote Validation: The program ensures that only valid votes are counted, helping prevent errors or invalid inputs.

2. Real-Time Vote Counting

Instant Feedback: The program provides immediate feedback after each vote, updating the vote count and informing the user that their vote has been recorded.

Dynamic Data Management: All votes are tallied and stored in variables during the program's execution, ensuring accurate tracking of the voting process.

3. Final Results Display

Summary of Votes: At the end of the voting process, the program displays the total number of votes each candidate received and the overall total votes cast. This summary is shown directly in the console.

4. CSV Export

Export Functionality: The program provides an option to export the final voting results to a CSV file. This feature is particularly useful for keeping a record of the election results or for further analysis.

Customizable Filename: Users can specify the name of the CSV file where the results will be saved, providing flexibility in data management.

5. User-Friendly Console Interface

Clear Instructions: The program guides users through the voting process with clear prompts and instructions, making it accessible to users of all experience levels.

Simple Navigation: Users can easily navigate the voting process, cast their votes, and finish the election with intuitive console commands.

Technical Implementation

The core logic of the program is contained within the cast_vote function, which checks the user's input, updates the respective candidate's vote count, and increments the total vote count. Usage Instructions

To run the Electoral Vote Simulator, follow these steps:

  1. Ensure Python is installed on your system.
  2. Clone the repository or download the script from https://github.com/jpb1991/python.
  3. Open a terminal or command prompt and navigate to the directory containing the main.py script.
  4. Run the script using the command: python main.py.
  5. Follow the on-screen instructions to cast votes for Candidate 1 or Candidate 2.
  6. Type end to finish voting and display the results.
  7. The results will be displayed in the console, and you can also save them to a CSV file by following the prompts.
