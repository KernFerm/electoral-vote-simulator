Electoral Vote Simulator Documentation

Project Overview

The Electoral Vote Simulator is a Python-based console application designed to simulate an electoral voting process between two candidates. Users can cast their votes through the console interface, and the final results are displayed at the end of the voting process. Additionally, the program allows the results to be exported to a CSV file for record-keeping purposes.

Purpose

This project aims to simulate a simple election scenario, enabling users to cast votes for two candidates. It is useful for demonstrating core Python programming skills, including input handling, conditional logic, and file operations. This project is ideal for educational purposes and showcases the application of basic Python concepts in a practical setting.

Key Features

1. Voting System

User Interaction: Users can cast votes for Candidate 1 or Candidate 2 via the console.

Vote Validation: The program ensures that only valid votes (for Candidate 1 or Candidate 2) are counted.

2. Real-Time Vote Counting

Immediate Feedback: Each vote is instantly tallied, and the user is notified that their vote has been recorded.

Data Management: The vote counts for each candidate are stored in variables during the program's execution.

3. Result Display and Export

End of Voting: When voting is concluded, the results are displayed in the console, showing the vote count for each candidate and the total votes cast.

CSV Export: The results can be exported to a CSV file (vote_results.csv) for record-keeping or further analysis.

Technical Implementation

Python Code Structure

The project consists of a single Python script (main.py) that is organized into the following functions:

  • cast_vote(vote, candidate_1_votes, candidate_2_votes, total_votes): This function handles vote casting, updating the vote counts based on user input.
  • display_results(candidate_1_votes, candidate_2_votes, total_votes): Displays the final results of the voting process in the console.
  • export_results_to_csv(candidate_1_votes, candidate_2_votes, total_votes, filename='vote_results.csv'): Exports the final vote counts to a CSV file.
  • main(): The main function that initializes the program, manages the voting loop, and calls the other functions as needed.

Code Explanation

The core logic of the program is contained within the cast_vote function, which increments the vote count for the selected candidate and updates the total votes. The display_results function is called at the end of the voting process to show the final tally, and the export_results_to_csv function allows the results to be saved to a CSV file for further use.

Usage Instructions

To run the Electoral Vote Simulator:

  1. Ensure you have Python installed on your system.
  2. Clone the repository or download the script from https://github.com/jpb1991/python.
  3. Open a terminal or command prompt and navigate to the directory containing the main.py script.
  4. Run the script using the command: python main.py
  5. Follow the on-screen prompts to cast votes for Candidate 1 or Candidate 2.
  6. Type end to finish voting and display the results.
  7. The results will be displayed in the console, and you can also find them saved in a file named vote_results.csv.
